建置時間估計 35-45 分鐘 ‧ 4 個 Notebook 共用、不分班 ‧ 完成後寫回每個 Notebook 的 Education Viewer URL。

0. 預備

用學校 Google 帳號登入 notebooklm.google.com(Education Viewer 分享需同網域)
確認 Chrome 或 Edge 桌機版,手機 / 平板 UI 不完整不要用
打開檔案總管到 C:\culturereports\pdf\,4 個事件子資料夾備用
不要開公開分享連結。只用 Education Viewer(read-only,同網域)。學生看得到內容、能問問題,但不能改 source。

1. 大港口事件 / Cepo' 戰役

步驟

NotebookLM 首頁 → 新增 Notebook
命名:花商二年級_大港口_2026v1
上傳兩支 PDF(從 pdf\大港口事件\):
  • 原民會叢書-大港口事件1877-1878.pdf
  • 國教院補充教材-大港口事件.pdf
等索引完成(右側 source 出現「Cited」前不要進下一步)
點 Audio Overview → Customize,Focus Prompt 貼下框文字後產生(僅老師上課示範用,學生不一定需要聽):
誰的土地被改變?強調命名差異(撫番 vs Cepo' 戰爭)與靜浦國小蓋在屠殺地的事實。
用學生開場問題驗證 — 在 chat 貼下框測試,看 NotebookLM 是否答得出來且引用合理:
請根據已上傳的大港口事件 PDF,整理清廷文獻稱此事為「事件 / 撫番 / 攻剿」,與阿美族稱 Cepo' 戰爭的差異,各列 3 點並附來源。
右上 Share → Education Viewer → 同校網域只讀分享
